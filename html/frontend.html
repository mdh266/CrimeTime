CrimeTime frontend package¶

This documentation of the frontend package of CrimeTime which uses Flask, HTML and CSS.

views module¶

Views module contains the Flask methods to be called on the front end and makes request to the various backend modules.

frontend.views.crime_time_output()[source]¶

This function drives the output for the website. It takes in the address supplied by the user and calls either of the methods,

  • frontend.all_crimes(...)
  • frontend.historical(...)
  • frontend.future(...)

Each of the above methods will then forward their results to a resulting html page.

Return type:html
