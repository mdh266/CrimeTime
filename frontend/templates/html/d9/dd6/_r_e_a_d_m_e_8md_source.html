1 **CRIMETIME**
2 
3 *Web application for visualizing and predict crime rates in New York City.*
4 
5 *To see the web application in action visit: www.crimetime.online.*
6 
7 The code was written in <a href="https://www.python.org/"> Python</a>
8 using <a href="http://flask.pocoo.org/"> Flask</a>.
9 and was deployed to <a href="https://aws.amazon.com/"> Amazon web services.</a>
10 
11 The crime data was provided by <a href="https://nycopendata.socrata.com/"> NYC Open Data</a>. The info for police precincts was obtained through scraping the NYPD's
12 <a href="http://www.nyc.gov/html/nypd/html/home/precincts.shtml"> website.</a> All data was cleaned and stored in a <a href="https://sqlite.org/">SQLite</a> database.
13 
14 Forecasted crime rates were predicted using a seasonal ARIMA model through the python
15 library <a href="http://statsmodels.sourceforge.net/"> Statsmodels</a>.
