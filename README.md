**CRIMETIME**

*Web application for visualizing crime rates in New York City.  To see the web application in action visit: www.crimetime.online.*

6.) Shapely (1.5.17)

8.) Flask (0.11.1)

9.) Basemap (1.0.7)

10.) Matplotlib (1.5.3)

12.) Numpy (1.11.2)
